Начальное условие: $$I\{u\} = \int\limits_{-1}^{1}u`^2 + u(2 + 2x) + u^2x^2\, dx, \quad u(-1) = 0, \quad u(1) = 2, \quad n = 2$$

Найдем значения \(\varphi_k:\)

$$\varphi_0 = 1 + x$$

$$\varphi_1 = C_1(1 + x)(-1 + x)$$

$$\varphi_2 = C_2x(1 + x)(-1 + x)$$

Разложим интеграл на составляющие: $$I\{u\} = \int\limits_{-1}^{1}u`^2\, dx + \int\limits_{-1}^{1}2u\, dx + \int\limits_{-1}^{1}u^2x^2\, dx + \int\limits_{-1}^{1}2ux\, dx $$

Возьмем интегралы по составляющим:

$$\int\limits_{-1}^{1}u`^2\, dx = \int\limits_{-1}^{1}1 + C_2^2 - 2C_2 - 6C_2^2x^2 + 4C_1x + 4C_1^2x^2 + 6C_2x^2 + 9C_2^2x^4 - 4C_1C_2x + 12C_1C_2x^3\, dx = 2 + \frac{8}{3}C_1^2 + \frac{8}{5}C_2^2$$

$$\int\limits_{-1}^{1}2u\, dx = \int\limits_{-1}^{1}2 - 2C_1 + 2x - 2C_2x + 2C_1x^2 + 2C_2x^3\, dx = 4 - \frac{8}{3}C_1$$

$$\int\limits_{-1}^{1}u^2x^2\, dx = \int\limits_{-1}^{1}x^2 + x^4 + 2x^3 + C_1^2x^2 + C_1^2x^6 + C_2^2x^4 + C_2^2x^8 - 2C_1x^2 - 2C_1x^3 - 2C_2x^3 - 2C_2x^4 - 2C_1^2x^4 - 2C_2^2x^6 + 2C_1x^4 + 2C_1x^5 + 2C_2x^5 + 2C_2x^6 - 4C_1C_2x^5 + 2C_1C_2x^3 + 2C_1C_2x^7\, dx = \frac{16}{15} - \frac{8}{15}C_1 - \frac{8}{35}C_2 + \frac{16}{105}C_1^2 + \frac{16}{315}C_2^2$$

$$\int\limits_{-1}^{1}2ux\, dx = \int\limits_{-1}^{1}2x + 2x^2 - 2C_1x - 2C_2x^2 + 2C_1x^3 + 2C_2x^4\, dx = \frac{4}{3} - \frac{8}{15}C_2$$

Найдем определенный интеграл данного выражения:

$$Ф(C_1, C_2) = I\{u\} = \frac{42}{5} - \frac{16}{5}C_1 - \frac{16}{21}C_2 + \frac{104}{63}C_2^2 + \frac{296}{105}C_1^2$$

Для выполнения условия экстремума необходимо решить следующию систему:

$$\begin{cases} \displaystyle \frac{\partial Ф}{\partial C_1} = -\frac{16}{5} + \frac{592}{105}C_1 = 0 \\ \displaystyle \frac{\partial Ф}{\partial C_2} = -\frac{16}{21} + \frac{208}{63}C_2 = 0 \end{cases}$$

Решенив данную систему мы получаем:

$$C_1 = \frac{21}{37}, C_2 = \frac{3}{13}$$

В общем виде выражение для экстремали примет вид:

$$Ф(C_1, C_2) = I\{u\} = \frac{16}{37} + \frac{3}{13}x^3 + \frac{10}{13}x + \frac{21}{37}x^2$$
