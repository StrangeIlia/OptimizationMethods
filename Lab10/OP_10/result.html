Найдем значения \(\varphi_k:\)

$$\varphi_0 = 1 + x$$

$$\varphi_1 = C_1(1 + x)(-1 + x)$$

$$\varphi_2 = C_2x(1 + x)(-1 + x)$$

$$Ф(C_1, C_2) = I\{u\} = \int\limits_{-1}^{1}1 + x^5 + x^7 - 1C_1 + 2x + 2x^2 + 2x^3 + 2x^6 + xC_2^2 + C_1^2x^5 + C_1^2x^9 + C_2^2x^7 + C_2^2x^11 - 6C_2^2x^3 - 3C_2x - 2C_1x^5 - 2C_1x^6 - 2C_2x^6 - 2C_2x^7 - 2C_1^2x^7 - 2C_2^2x^9 + 2C_1x^4 + 2C_1x^7 + 2C_1x^8 + 2C_2x^5 + 2C_2x^8 + 2C_2x^9 + 3C_1x^2 + 4C_1^2x^3 + 5C_2x^3 + 9C_2^2x^5 - 4C_1C_2x^2 - 4C_1C_2x^8 + 2C_1C_2x^6 + 2C_1C_2x^10 + 12C_1C_2x^4\, dx$$

Найдем определенный интеграл данного выражения:

$$Ф(C_1, C_2) = I\{u\} = \frac{82}{21} - \frac{8}{63}C_2 + \frac{212}{315}C_1 + \frac{7552}{3465}C_1C_2$$

Для выполнения условия экстремума необходимо решить следующию систему:

$$\begin{cases} \displaystyle \frac{\partial Ф}{\partial C_1} = \frac{212}{315} + \frac{7552}{3465}C_2 = 0 \\ \displaystyle \frac{\partial Ф}{\partial C_2} = -\frac{8}{63} + \frac{7552}{3465}C_1 = 0 \end{cases}$$

Решенив данную систему мы получаем:

$$C_1 = \frac{55}{944}, C_2 = -\frac{583}{1888}$$
